## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+T | 新建标签 |
| Ctrl+W | 关闭当前标签 |
| Ctrl+Tab | 下一个标签 |
| Ctrl+Shift+Tab | 上一个标签 |
| Ctrl+1-9 | 快速跳转到第 N 个标签 |

## 📝 下一步: Step 3 - 状态绑定

### 需要实现的功能

1. **终端输出绑定**
   - 每个标签显示自己的终端输出
   - 切换标签时更新终端显示

2. **SSH 会话绑定**
   - 每个标签维护独立的 SSH 会话
   - 连接状态独立管理

3. **命令输入绑定**
   - 每个标签记住自己的命令输入

4. **AI 对话绑定**
   - 每个标签独立的 AI 对话历史
